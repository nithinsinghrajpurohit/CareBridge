# CareBridge

Full Stack Application leveraging **React (Vite)** for intelligent community task routing and priority prediction.

## Folder Overview
- `/frontend`: React application for Requesters and Volunteers.

---

## Frontend Setup (React)

### Prerequisites
- Node.js (v18+)

### Steps
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install standard dependencies:
   ```bash
   npm install
   ```
3. Install new required packages for full-stack integration:
   ```bash
   npm install axios react-router-dom react-leaflet leaflet jwt-decode
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
